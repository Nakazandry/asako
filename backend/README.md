# ASAKO Backend

API REST Express pour la gestion des offres, candidatures et entretiens.

## Demarrage local

```bash
cp .env.example .env
npm install
npm run dev
```

Variables importantes pour Render:

- `PORT`
- `CLIENT_URL`
- `DB_HOST`
- `DB_PORT`
- `DB_NAME`
- `DB_USER`
- `DB_PASSWORD`
- `JWT_SECRET`
- `JWT_EXPIRES_IN`
